# STA-141b-Final-Project-Airbnb
Exploring Airbnb data

# How to run code:
1. Download dataset from https://www.kaggle.com/datasets/mysarahmadbhat/airbnb-listings-reviews 
2. Extract the dataset and add the "Airbnb Data" folder to the same directory as the python notebooks
3. Run "data_cleaning+visuals.ipynb"
4. Now you are able to run all the other notebooks
